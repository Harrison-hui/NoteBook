    元组型¶

    linestyle=(offset, on_off_seq)

    1. offset：在构建的线型上开始的偏移量（以点为单位）

    2. on_off_seq：通过序列构建线型。若序列长度为偶数，则偶数位(0开始)代表可见线段长度（以点为单位），奇数位表示不可见线段的长度（以点为单位）；若序列长度为奇数，则序列扩增为2倍【即(3, 1, 1) 相当于 (3, 1, 1, 3, 1, 1)】

    Path创建(Markers created from Paths)¶

    Marker reference — Matplotlib 3.10.1 documentation

    绘制坐标以原始标记点为原点(0, 0)

    1. mpath.Path(vertices, codes=None, closed=False)

      1. vertices：绘制的点坐标序列（如[(x1, y1), (x2, y2), ...]）或者形状(N, 2)的数组（如[[x1 y1] [x2 y2] ...]）。

      2. codes：与vertices每个坐标一一对应，表示到该坐标的动作，可选：

        1. mpath.Path.STOP = np.uint8(0) = 0：不绘制该坐标且结束绘制

        2. mpath.Path.MOVETO = np.uint8(1) = 1：不绘制，仅移动到该坐标。第一个必须是该值。

        3. mpath.Path.LINETO = np.uint8(2) = 2：以线段绘制到该坐标

        4. mpath.Path.CURVE3 = np.uint8(3) = 3：绘制曲线到该坐标后一个坐标

        5. mpath.Path.CURVE4 = np.uint8(4) = 4：绘制曲线到该坐标后两个坐标

        6. mpath.Path.CLOSEPOLY = np.uint8(79) = 79：不绘制该坐标，直接前一个坐标绘制线段到第一个坐标以封闭图形

      3. closed：是否封闭图形。当codes=None且closed=True时，相当于codes=[1, 2, 2, ..., 79]。

    2. mpath.Path.arc(theta1, theta2, n=None, is_wedge=False)：绘制从 theta1°到 theta2°（>360°时将减去360°）的单位圆弧。n，略。 is_wedge 指是否绘制弧形半径。

    3. mpath.Path.wedge(theta1, theta2, n=None)：相当于mpath.Path.arc中 is_wedge=True

    4. mpath.Path.circle(center=(0.0, 0.0), radius=1.0, readonly=False)：绘制圆心坐标为 center，半径为 radius 的圆。readonly，略。

    5. mpath.Path.unit_circle()：绘制单位圆

    6. mpath.Path.unit_circle_righthalf()：绘制单位圆右侧半

    7. mpath.Path.unit_rectangle()：绘制单位矩形（正方形）

    8. mpath.Path.unit_regular_asterisk(numVertices)：绘制单位正 numVertices 边星

    9. mpath.Path.unit_regular_polygon(numVertices)：绘制单位正 numVertices 边形

    10. mpath.Path.unit_regular_star(numVertices, innerCircle=0.5)：绘制单位正 numVertices 角星。innerCircle，略
